# yapblog

`yapblog` is acronym for `yet another personal blog`.

# Requirement

Please install `python3`, `flask`,`flask-login`, `flask-sqlalchemy`.

# `/api`
## `/api/user`
与单个用户相关的接口，参考 `yapblog.api.user` ：

+ `/api/user`
+ `/api/user/register`
+ `/api/user/login`
+ `/api/user/logout`

# `/register`
注册页面。

# `/login`
登录页面。

# `/logout`
登出页面。

# `/user`
用户信息页面。
